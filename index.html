Project 1

Automated Plant
Watering System

The goal of the project is to implement an automatic plant watering system using a Raspberry Pi 3, a moisture sensor, a T-cobbler, an Arduino relay module, a breadboard, and a water pump. By connecting these components and synchronizing them with code written in the Thonny IDE environment, the system will use the sensor to determine whether the soil is moist or dry and, based on this, activate the water pump.
